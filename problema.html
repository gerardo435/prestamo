PROBLEMA

Desarrolle una aplicación web que resuelva el siguiente problema.
Importante en el formulario debe solicitar los siguientes datos:
1. Datos del cliente. (nombre, apellido, edad)
2. Datos laborales (nombre de la empresa, tiempo, tipo de empresa (privada o pública)).
3. Datos del préstamo.
a. Tipo de préstamo (personal, estudio, hipotecario o consumo).
b. Banco con el que desea realizar el préstamo:

Banco
Agrícola
CONDICIONES PARA REALIZAR EL PRESTAMO
✓ Si pide el préstamo es para más de cinco años la tasa de interés que se le cobrará
es del 12%, caso contrario la tasa será del 10%.
✓ No importa el monto que preste.
✓ En caso de ser un empleado público, las tasas se disminuyen en un 2%.

Banco Azul CONDICIONES PARA REALIZAR EL PRESTAMO
✓ Banco azul solo ofrece como tiempo máximo para que usted pague la deuda
de 1 a 5 años, y el monto máximo es de $120000.
✓ Para toda la tasa es del 12%
✓ En caso de ser un empleado público la tasa de interés que se le cobrará será
del 9%.

Banco
Cuscatlán
CONDICIONES PARA REALIZAR EL PRESTAMO
El banco no ofrece créditos hipotecarios.
Solo le presta 12 veces su salario.
La tasa de interés que cobra es del 19%.
No tiene restricciones para el plazo.


c. El plazo o tiempo que necesita para cancelar el crédito.

1. Cantidad de $$$ en concepto de intereses ( es decir la ganancia del banco)
